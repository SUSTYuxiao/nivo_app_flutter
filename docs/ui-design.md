## InMeet 实时会议

三个状态：idle → recording → result

### Idle 状态
- 居中大标题"实时会议"
- 小字"点击下方按钮开始录音和实时转录"
- 高精度去噪开关（默认关闭）
  - 关闭：`.default` 模式，能录到扬声器声音
  - 开启：`.voiceChat` 模式，过滤杂音和扬声器
- "发起会议"按钮

### Recording 状态
- 录音脉冲动画 + 计时器
- "实时转录中"标签
- 转录文本滚动列表（partial 原地更新，final 新增行）
- "结束会议"按钮 → 暂停计时器 → 弹出行业/模板选择弹窗
  - 确认：结束录音 + 生成纪要
  - 取消：恢复计时器继续录音

### Result 状态
- Markdown 渲染纪要
- "返回"按钮回到 idle

## AfterMeet 会后整理

两个状态：idle → result

### Idle 状态
- 居中大标题"会后整理"
- 小字"选择文件开始会后整理"
- "开始整理"按钮 → 弹出 modal

### Modal（底部弹窗）
- 会议文本输入（可选，多行 TextField）
- 已选文件列表（带移除按钮）
- 本地录音列表（来自 InMeet 的录音，可选择、可删除）
- "从文件选择"按钮（FileType.any）
- "从录音机同步"按钮 → 弹出指引 dialog
- "提交整理"按钮 → 弹出行业/模板选择 → 离线转写音频 → chatRun

### Result 状态
- Markdown 渲染结果
- "新整理"按钮重置

## 共享组件

### IndustryTemplateDialog（底部弹窗）
- 行业 Wrap chips（企业服务/金融/消费/科技/制造）
- 模板 Wrap chips（自定义/深度纪要/对话式/关键点/任务分配）
- "确认"按钮
- InMeet 结束会议和 AfterMeet 提交都复用

## 设置页

分组卡片风格：
- 语音识别：ASR 模式切换（自动/本地）+ Nivo 云端转写开关（仅自动模式显示）
- 本地模型：仅 Android + 本地模式显示模型下载
- iOS 本地模式：显示"使用 iOS 系统语音识别，无需下载模型"
- 账户：退出登录
- 关于：版本号

## 历史记录页

- 大标题"会议历史"
- 时间筛选 chips（今天/7天内/30天内）
- 白色卡片列表（标题 + 时间 + 行业标签）
- 下拉刷新 + 上拉加载更多
- 点击进入详情页（元信息 + 纪要 Markdown 渲染 + 改标题/删除）

## iOS 语音备忘录导入指引

用户无法直接从文件选择器访问语音备忘录（iOS 沙盒限制）。
指引步骤：
1. 打开「语音备忘录」App
2. 长按录音，点击「分享」
3. 选择「存储到文件」
4. 回到 Nivo，点击「从文件选择」
